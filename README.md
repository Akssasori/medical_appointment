# Dependencias

npm i express
npm add typescript ts-node nodemon -D

## precissei rodar esta comando quando adicionei o express
npm i --save-dev @types/express

npx tsc --init

-- criar na raiz o arquivo nodemon.json

### start app

npm run dev
